## Qué es este documento

La Product Bible es la fuente oficial de verdad del producto All Motors Cloud.

Toda decisión funcional, filosófica y de diseño queda registrada aquí.

Cuando exista contradicción entre la documentación y el código:
**la documentación tiene prioridad. El código se adapta.**

---

## Cómo usar esta documentación

- Leer antes de implementar cualquier funcionalidad.
- Consultar antes de tomar decisiones de arquitectura.
- Actualizar cuando una decisión cambie.
- No implementar lo que no esté documentado aquí.
