# Roadmap — All Motors ERP

Estado: vigente al 2026-06-30.

Este roadmap define las versiones planificadas del ERP y los módulos incluidos en cada una.
Se actualiza a medida que el proyecto avanza. Toda versión lanzada se documenta en CHANGELOG.md.

---

## Versión 1 — Núcleo Operativo
*Objetivo: el taller puede operar digitalmente sin papel desde el primer día.*

- Recepción de vehículos (OT con datos completos, firma digital de recepción)
- Gestión de vehículos (ficha, historia técnica, alertas de mantención)
- Gestión de clientes (ficha, historial, tipos: particular / empresa / aseguradora)
- Orden de Trabajo completa (estados, diagnóstico, trabajos, repuestos)
- Catálogo de servicios (servicios, paquetes, tarifas)
- Inventario básico (stock, alertas de mínimo, recepción de mercadería)

---

## Versión 2 — Operación Financiera
*Objetivo: el taller gestiona su ciclo de cobro completo y controla garantías.*

- Presupuestos (construcción automática, aprobación del cliente, vencimiento)
- Facturación (boleta y factura electrónica SII, nota de crédito, cobros)
- Garantías (nacimiento automático, proceso de reclamo, garantía de proveedor)
- CRM básico (ficha unificada, recordatorios de mantención, seguimiento post-entrega)
- Dashboard (KPIs del día, semana y mes)

---

## Versión 3 — Automatización e Inteligencia
*Objetivo: el sistema trabaja solo en los procesos repetibles.*

- IA integrada (sugerencias de servicios según historial, estimaciones de tiempo, alertas predictivas)
- Workflow configurable (flujos de aprobación parametrizables por el administrador)
- Automatizaciones (mensajes automáticos al cliente, reposición de stock, recordatorios de cobro)
- Portal Cliente (estado del vehículo en tiempo real, aprobación de presupuestos, historial propio)
- Módulo de subcontratos (vinculación de facturas de terceros con OTs)

---

## Versión 4 — Plataforma y Escala
*Objetivo: All Motors ERP como plataforma extensible para otros talleres.*

- Marketplace (repuestos y servicios entre talleres de la red)
- API pública (integración con plataformas externas: seguros, proveedores, SII)
- Multiempresa avanzada (gestión centralizada de múltiples talleres con reportes consolidados)
- Aplicación móvil (interfaz para mecánicos en tablet, seguimiento de OTs para clientes)
- Integración WhatsApp Business API (aprobaciones, notificaciones, respuestas automáticas)
- Sistema de comisiones (cálculo automático de comisiones para mecánicos por producción)

---

## Notas de planificación

- Cada versión debe ser funcional de manera autónoma antes de iniciar la siguiente.
- Las versiones no tienen fechas fijas: se completan cuando están operativamente correctas.
- Toda decisión que modifique el alcance de una versión debe actualizarse en este documento y registrarse en CHANGELOG.md.
- La referencia funcional de cada versión es ERP_MASTER_v1.0.md (y sus versiones futuras).
